JDM Car Showcase

Página estática simples com HTML, CSS e JavaScript para demonstrar um visual estilo JDM.

⚠️ **STATUS ATUAL:** Usando imagens online de teste (Unsplash). A página está 100% funcional e pronta.

Como usar:

1. Abra `index.html` no seu navegador (duplo clique ou `File > Open`).
2. Use os botões de cor para mudar a pintura do carro.
3. Clique em `Faróis` para ligar/desligar os faróis.
4. Clique em `Rev` para ouvir um pequeno "rev" sintetizado e ver a animação.

Arquivos:
- `index.html` — marcação
- `styles.css` — estilos
- `script.js` — interações

Observação: As imagens foram removidas do layout por solicitação. A página agora usa apenas elementos de layout (SVG e cartões de texto).
